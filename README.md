1. Simulated the working of an ATM System using ReactJS, JavaScript, with features like user login, balance check, deposit, withdrawal, and transaction history.
2. Used core Data Structures like Map and Array.
3. Built the frontend using ReactJS. Used the useState and useRef hooks to store the values entered by the user.
4. Used JSON files to keep track of the account balance and transaction history of the users.
5. Used Express JS in the backend to update the account balance of the user and store the transaction info after performing transactions like deposit and withdrawal.
